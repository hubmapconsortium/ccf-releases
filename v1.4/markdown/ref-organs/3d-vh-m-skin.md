# 3D Reference Organ for Skin, Male v1.2

### Description
[This reference organ](https://hubmapconsortium.github.io/ccf/pages/ccf-3d-reference-library.html) was created using data from the Visible Human Male, provided by the National Library of Medicine. It was adjusted to add space for the triceps surae and the ankle bones for the eventual addition of the triceps surae muscles.
