# 3D Reference Organ for Brain, Male v1.0

### Description
[This reference organ](https://humanatlas.io/3d-reference-library) was created using data from the "[Allen Human Reference Atlas â€“ 3D, 2020](https://doi.org/10.1002/cne.24080)" representing one half of the human brain (Ding et al., 2016).
